5. Define or explain the following:
(a)Standards    (b) Units    (c) SI system    (d) CGS system    (e)British system

(a) Standards
The standards are certain agreed quantity of some kind referred to as a unit.

(b)Units
A unit is a perticular physical quantity, defined and adopted by convention, with which other particular quantity of the same kind are compared to express their values.

(c)SI system
SI system is the international system of units.It is the modified form of the MKS system of unit. It uses metre, kilogram, second as basic units.

(d)CGS system
The system which uses centimetre, gram and second as a basic unit is called CGS system.

(e)British system(FPS system)
The system which uses foot, pound, and second as basic units is called British system.

6. The SI system has been accepted because it has many advantages compared to the previous systems uesd.Discuss these advantages.
တစ္ျခားစနစ္ေတြနဲ႔ ႏိႈင္းလိုက္ရင္ အားသာခ်က္ေတြ ႐ွိလို႔ SI စနစ္ကို လက္ခံတာတဲ့။ အဲ့ဒီအားသက္ခ်က္ေတြက ဘာေတြလဲတဲ့။

Advantages of SI system are -
(1) It is a coherent system which bases on a certain set of basic units.
(2) It is a rational system as it assigns only one unit to a particular physical quantity.
(3) It is a absolute system of unit.
(4) It is a meteric system and so multiples and sub multiples of units are express as power of 10.

သူရဲ႕ အားသက္ခ်က္ေတြက ႐ိုး႐ွင္းလြယ္ကူတဲ့စနစ္ (coherent system and rational system) ျဖစ္တယ္။ ၿပီးျပည့္စံုတဲ့ စနစ္ (absolute system) ျဖစ္တယ္။ meteric စနစ္ျဖစ္လို႔ ၁၀ ပါဝါနဲ႔ တန္ဖိုးေတြကို အတုိးအေလ်ာ့လုပ္လို႔ရတယ္။
